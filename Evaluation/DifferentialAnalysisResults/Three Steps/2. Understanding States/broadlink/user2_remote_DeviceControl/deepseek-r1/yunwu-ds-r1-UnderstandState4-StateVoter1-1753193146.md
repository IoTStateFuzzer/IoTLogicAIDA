# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一实例）；user2未收到邀请，不是家庭成员，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第一实例）并邀请user2到家庭；user2收到了邀请但未同意，不是家庭成员，无控制权限。|
|4|user1邀请user2到家庭但未添加设备；user2收到了邀请但未同意，不是家庭成员，无控制权限。|
|5|user1添加了设备（第一实例）并邀请；user2扫描了QR码但未接受邀请，不是家庭成员，无控制权限。|
|6|user1添加了设备（第一实例）并邀请user2为家庭成员；user2已同意邀请，是家庭成员，拥有控制权限。|
|7|user1添加了设备（第一实例）并多次邀请；user2扫描QR码并接受邀请，是家庭成员，拥有控制权限。|
|8|user1邀请user2到家庭但未添加设备；user2已同意邀请，是家庭成员，但无设备控制权。|
|9|user1多次邀请user2到家庭但未添加设备；user2已同意邀请，是家庭成员，但无设备控制权。|
|10|user1邀请user2到家庭但未添加设备；user2扫描了QR码但未接受邀请，不是家庭成员，无控制权限。|
|11|user1添加了设备（第一实例）并邀请；user2曾是家庭成员但已退出家庭，权限被撤销，无控制权限。|
|12|user1邀请user2到家庭但未添加设备；user2曾是家庭成员但已退出，权限被撤销，无控制权限。|
|13|user1多次邀请user2到家庭但未添加设备；user2扫描了QR码但未接受邀请，不是家庭成员，无控制权限。|
|14|user1添加了设备（第一实例）并多次邀请；user2扫描了QR码但未接受邀请，不是家庭成员，无控制权限。|
