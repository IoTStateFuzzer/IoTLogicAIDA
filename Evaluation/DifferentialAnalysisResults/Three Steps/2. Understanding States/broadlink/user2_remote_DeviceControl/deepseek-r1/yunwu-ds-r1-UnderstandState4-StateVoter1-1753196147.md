# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第1个实例）；user2未收到邀请，不是家庭成员，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第1个实例）并发送邀请给user2；user2收到邀请未同意，不是家庭成员，无控制权限。|
|4|user1未添加设备但发送邀请给user2；user2收到邀请未同意，不是家庭成员，无控制权限。|
|5|user1已添加设备（第1个实例）并发送邀请；user2扫描二维码但未接受邀请，收到邀请未同意，不是家庭成员，无控制权限。|
|6|user1已添加设备（第1个实例）并发送邀请；user2已接受邀请，是家庭成员，有控制权限。|
|7|user1已添加设备（第1个实例）并发送邀请；user2已接受邀请，是家庭成员，有控制权限。|
|8|user1未添加设备但发送邀请；user2已接受邀请，是家庭成员，但无设备，因此无控制权限。|
|9|user1未添加设备但发送邀请；user2已接受邀请，是家庭成员，但无设备，因此无控制权限。|
|10|user1未添加设备但发送邀请；user2扫描二维码但未接受邀请，收到邀请未同意，不是家庭成员，无控制权限。|
|11|user1已添加设备（第1个实例）并发送邀请；user2曾接受但已退出家庭，不再是家庭成员，无控制权限。|
|12|user1未添加设备但发送邀请；user2曾接受但已退出家庭，不再是家庭成员，无控制权限。|
|13|user1未添加设备但发送邀请；user2扫描二维码但未接受邀请，收到邀请未同意，不是家庭成员，无控制权限。|
|14|user1已添加设备（第1个实例）并发送邀请；user2扫描二维码但未接受邀请，收到邀请未同意，不是家庭成员，无控制权限。|
