# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一个实例）；user2未收到邀请，无控制权。|
|2|错误状态|
|3|user1已添加设备（第一个实例）并邀请user2；user2收到邀请未同意，不是家庭成员，无控制权。|
|4|user1未添加设备，但邀请user2；user2收到邀请未同意，不是家庭成员，无控制权。|
|5|user1已添加设备（第一个实例）并邀请user2；user2收到邀请未同意，不是家庭成员，无控制权。|
|6|user1已添加设备（第一个实例）；user2已同意邀请，是家庭成员，有控制权。|
|7|user1已添加设备（第一个实例）；user2是家庭成员，有控制权。|
|8|user1未添加设备，但邀请user2；user2是家庭成员，有控制权。|
|9|user1未添加设备；user2是家庭成员，有控制权。|
|10|user1未添加设备，邀请user2；user2收到邀请未同意，不是家庭成员，无控制权。|
|11|user1已添加设备（第一个实例）；user2曾是家庭成员但已退出，不再是家庭成员，无控制权。|
|12|user1未添加设备；user2不再是家庭成员，无控制权。|
|13|user1未添加设备，邀请user2；user2收到邀请未同意，不是家庭成员，无控制权。|
|14|user1已添加设备（第一个实例）并邀请user2；user2收到邀请未同意，不是家庭成员，无控制权。|
