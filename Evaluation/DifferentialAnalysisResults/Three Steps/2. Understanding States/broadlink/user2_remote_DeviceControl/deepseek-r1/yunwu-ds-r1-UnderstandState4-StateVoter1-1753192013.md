# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一个实例）；user2未收到邀请，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第一个实例）并邀请user2为家庭成员；user2已收到邀请但未同意。|
|4|user1未添加设备但邀请user2为家庭成员；user2已收到邀请但未同意。|
|5|user1已添加设备（第一个实例）并邀请user2；user2已扫描二维码但未接受邀请。|
|6|user1已添加设备（第一个实例）；user2是家庭成员，权限状态为“有控制权”。|
|7|user1已添加设备（第一个实例）；user2是家庭成员，权限状态为“有控制权”。|
|8|user1未添加设备；user2是家庭成员，权限状态为“有控制权”。|
|9|user1未添加设备；user2是家庭成员，权限状态为“有控制权”。|
|10|user1未添加设备但邀请user2；user2已扫描二维码但未接受邀请。|
|11|user1已添加设备（第一个实例）；user2曾接受邀请但已退出家庭，权限状态为“无控制权”。|
|12|user1未添加设备；user2曾接受邀请但已退出家庭，权限状态为“无控制权”。|
|13|user1未添加设备但多次邀请user2；user2已扫描二维码但未接受邀请。|
|14|user1已添加设备（第一个实例）并多次邀请user2；user2已扫描二维码但未接受邀请。|
