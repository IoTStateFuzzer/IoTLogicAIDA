# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第1个实例）；user2未收到邀请，没有控制权限。|
|2|错误状态|
|3|user1已添加设备（第1个实例）并邀请user2作为家庭成员；user2收到邀请未同意，不是家庭成员，没有控制权限。|
|4|user1未添加设备，但邀请user2作为家庭成员；user2收到邀请未同意，不是家庭成员，没有控制权限。|
|5|user1已添加设备（第1个实例）并邀请user2；user2扫描了QR码但未接受邀请，收到邀请未同意，不是家庭成员，没有控制权限。|
|6|user1已添加设备（第1个实例）并邀请user2；user2已接受邀请，成为家庭成员，有控制权限。|
|7|user1已添加设备（第1个实例）并多次邀请user2；user2已接受邀请，成为家庭成员，有控制权限。|
|8|user1未添加设备，但邀请user2；user2已接受邀请，成为家庭成员，但由于没有设备，没有控制权限。|
|9|user1未添加设备，但多次邀请user2；user2已接受邀请，成为家庭成员，没有控制权限（无设备）。|
|10|user1未添加设备，但邀请user2；user2扫描了QR码但未接受邀请，收到邀请未同意，不是家庭成员，没有控制权限。|
|11|user1已添加设备（第1个实例）并邀请user2；user2曾接受邀请成为家庭成员但已退出家庭，不再是家庭成员，没有控制权限。|
|12|user1未添加设备，但邀请user2；user2曾接受邀请成为家庭成员但已退出家庭，不再是家庭成员，没有控制权限。|
|13|user1未添加设备，但多次邀请user2；user2扫描了QR码但未接受邀请，收到邀请未同意，不是家庭成员，没有控制权限。|
|14|user1已添加设备（第1个实例）并多次邀请user2；user2扫描了QR码但未接受邀请，收到邀请未同意，不是家庭成员，没有控制权限。|
