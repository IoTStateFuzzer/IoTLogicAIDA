# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加设备（第一次添加设备），尚未进行共享；user2未收到任何邀请，暂无控制权限。|
|s2|错误状态。|
|s3|user1已添加设备（第一次添加设备）并执行了共享操作；user2自动获得对该设备实例的直接控制权限（未作为家庭成员）。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加设备（第一次添加设备实例），尚未进行共享；user2未收到邀请，暂无控制权限。|
|s2|错误状态。|
|s3|user1已添加设备（设备实例#1）并执行了共享操作；user2自动获得该设备的直接控制权限。|
|s4|user1添加设备并共享后，user2执行了设备控制操作，仍使用设备实例#1的直接共享权限。|
|s5|user1在设备实例#1上共享后，user2曾控制设备，但随后user1取消了共享，导致user2的控制权限被撤销。|
|s6|user1在设备实例#1上共享并允许user2控制后，又将该设备移除，导致设备实例和共享权限均被取消。|
|s7|user1先添加、共享并移除设备实例#1后，再添加一个新设备（设备实例#2），但未进行共享；user2因此不具备控制权限。|
|s8|user1添加新设备实例#2后执行了共享操作，user2自动获得该新设备实例的直接控制权限。|
