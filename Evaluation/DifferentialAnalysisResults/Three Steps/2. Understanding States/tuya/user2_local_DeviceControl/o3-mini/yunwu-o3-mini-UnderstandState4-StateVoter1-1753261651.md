# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1成功添加了第1个设备实例，但未进行共享操作；user2没有获得任何设备控制权限。|
|s2|错误状态。|
|s3|user1成功添加了第1个设备实例并执行了共享操作；user2通过直接共享方式自动获得该设备的控制权限（非家庭成员身份）。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已成功添加设备（第1个设备实例），未进行共享；user2未获得设备控制权限。|
|s2|错误状态。|
|s3|user1添加了第1个设备实例并进行了共享操作；user2自动获得该设备的控制权限。|
|s4|在添加并共享第1个设备实例后，user2执行了设备控制操作，依旧拥有设备控制权限。|
|s5|user1在第1个设备实例共享后撤销了共享；因此user2失去对该设备的控制权限。|
|s6|user1移除了第1个设备实例；user2的控制权限因设备移除而失效。|
|s7|在移除第1个设备实例后，user1重新添加了设备，形成第2个设备实例，但未进行共享；user2无控制权限。|
|s8|user1对第2个设备实例执行了共享操作；user2随即自动获得设备控制权限。|
