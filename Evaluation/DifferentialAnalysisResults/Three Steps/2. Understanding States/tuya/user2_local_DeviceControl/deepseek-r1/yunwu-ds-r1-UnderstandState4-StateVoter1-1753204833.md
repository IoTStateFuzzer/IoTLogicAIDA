# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已成功添加设备（第一个实例）；user2未收到邀请，不是家庭成员，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第一个实例）并直接共享；user2已获得设备控制权限（直接共享方式），无需显式接受。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已成功添加设备（第一个实例）；user2未收到邀请，不是家庭成员，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第一个实例）并直接共享；user2已获得设备控制权限（直接共享方式），无需显式接受。|
|4|user1已添加设备（第一个实例）并直接共享；user2已获得设备控制权限（直接共享方式），无需显式接受。|
|5|user1已添加设备（第一个实例）并撤销共享；user2不再具有控制权限（直接共享被撤销）。|
|6|user1已移除设备（第一个实例）；user2不再具有控制权限（设备移除导致共享失效）。|
|7|user1已重新添加设备（第二个实例）；user2未获得共享邀请，无控制权限。|
|8|user1已重新添加设备（第二个实例）并直接共享；user2已获得设备控制权限（直接共享方式）。|
