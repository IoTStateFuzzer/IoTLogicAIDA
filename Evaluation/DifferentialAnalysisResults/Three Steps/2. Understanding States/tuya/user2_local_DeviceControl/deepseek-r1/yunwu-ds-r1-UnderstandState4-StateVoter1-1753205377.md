# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1 已成功添加设备（第1个实例）；user2 未收到邀请，无权限。|
|2|错误状态|
|3|user1 已添加设备（第1个实例）并共享设备；user2 具有控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1 已成功添加设备（第1个实例）；user2 未收到邀请，无权限。|
|2|错误状态|
|3|user1 已添加设备（第1个实例）并共享设备；user2 具有控制权限（直接共享）。|
|4|user1 设备已添加并共享（第1个实例）；user2 具有控制权限（直接共享）。|
|5|user1 设备已添加但未共享；user2 无控制权限。|
|6|user1 无设备；user2 无权限。|
|7|user1 已重新添加设备（第2个实例）；user2 未共享，无权限。|
|8|user1 设备已重新添加并共享（第2个实例）；user2 具有控制权限（直接共享）。|
