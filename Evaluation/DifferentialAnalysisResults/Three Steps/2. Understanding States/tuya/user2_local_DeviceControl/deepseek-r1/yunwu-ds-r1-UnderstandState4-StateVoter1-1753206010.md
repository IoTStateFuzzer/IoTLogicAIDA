# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已成功添加设备（第1次）；user2未收到邀请，没有控制权限。|
|2|错误状态|
|3|user1已添加设备（第1次）并共享；user2有控制权限（直接共享）。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已成功添加设备（第1次）；user2未收到邀请，没有控制权限。|
|2|错误状态|
|3|user1已添加设备（第1次）并共享；user2有控制权限（直接共享）。|
|4|user1已添加设备（第1次）并共享；user2有控制权限（直接共享）。|
|5|user1已添加设备（第1次）但取消共享；user2没有控制权限（权限已撤销）。|
|6|user1设备已移除；user2没有控制权限（权限失效）。|
|7|user1已重新添加设备（第2次）；user2未收到邀请，没有控制权限。|
|8|user1已添加设备（第2次）并共享；user2有控制权限（直接共享）。|
