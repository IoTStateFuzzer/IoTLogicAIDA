# Base Model
|状态|语义描述|
|-----|---------|
|S0|初始状态|
|S1|user1已添加设备（第一次）；user2未收到邀请，无控制权限。|
|S2|错误状态|
|S3|user1已添加设备（第一次）并共享设备；user2已获得控制权限（直接共享）。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|S0|初始状态|
|S1|user1已添加设备（第一次）；user2未收到邀请，无控制权限。|
|S2|错误状态|
|S3|user1已添加设备（第一次）并共享设备；user2已获得直接共享的控制权限。|
|S4|user1已添加设备（第一次）并共享设备；user2拥有直接共享的控制权限。|
|S5|user1已添加设备（第一次）但取消共享；user2无控制权限（权限已撤销）。|
|S6|user1已移除设备；user2无控制权限（设备移除导致权限失效）。|
|S7|user1已重新添加设备（第二次）；user2无控制权限（新设备实例需重新共享）。|
|S8|user1已重新添加设备（第二次）并共享；user2再次获得直接共享的控制权限。|
