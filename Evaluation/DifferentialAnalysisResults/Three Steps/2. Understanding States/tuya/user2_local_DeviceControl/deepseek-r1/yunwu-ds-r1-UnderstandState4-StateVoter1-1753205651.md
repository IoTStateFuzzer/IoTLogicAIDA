# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已成功添加设备（第一次实例）；user2无权限。|
|2|错误状态|
|3|user1已添加设备并共享给user2；user2拥有控制权限（基于直接共享）。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一次实例）；未共享，user2无权限。|
|2|错误状态|
|3|user1已添加设备（第一次实例）并直接共享；user2拥有控制权限（直接共享）。|
|4|user1设备状态同S3；user2成功行使控制权限（权限未变更）。|
|5|user1设备状态同S3但已取消共享；user2权限被撤销。|
|6|user1已移除设备；user2权限失效（设备实例消失）。|
|7|user1已重新添加设备（第二次实例）；未共享，user2无权限。|
|8|user1已添加设备（第二次实例）并直接共享；user2重新获得控制权限。|
