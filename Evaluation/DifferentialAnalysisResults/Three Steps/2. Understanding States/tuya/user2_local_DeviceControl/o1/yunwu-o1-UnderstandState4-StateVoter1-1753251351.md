# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已成功添加设备；user2尚未收到邀请，无控制权限。|
|2|错误状态|
|3|user1已成功添加并共享设备；user2通过直接共享自动获得该设备的控制权限。|

# Divergent Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第一个设备实例；user2未收到邀请，无控制权限。|
|2|错误状态|
|3|user1已添加并直接共享设备；user2通过共享自动获得该设备控制权限。|
|4|user1已添加并共享设备；user2成功控制设备。|
|5|user1取消了对该设备的共享；user2失去设备控制权限。|
|6|user1移除了设备；设备被移除后user2不再拥有控制权限。|
|7|user1再次添加新设备（第二个实例）；暂未对user2进行共享，user2无控制权限。|
|8|user1在第二个实例中共享设备；user2通过直接共享自动获得该设备控制权限。|
