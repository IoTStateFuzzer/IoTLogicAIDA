# Base Model
|状态|语义描述|
|---|---|
|s0|初始状态|
|s1|user1已添加第1个设备；user2未被共享，无法控制设备。|
|s2|错误状态|
|s3|user1已添加第1个设备并直接共享给user2；user2因自动获得权限而可控制该设备。|

# Divergent Model
|状态|语义描述|
|---|---|
|s0|初始状态|
|s1|user1已添加第1个设备；user2未被共享，无法控制设备。|
|s2|错误状态|
|s3|user1已添加第1个设备并直接共享给user2；user2因自动获得权限而可控制设备。|
|s4|user1已添加第1个设备并共享给user2；user2正在进行设备控制。|
|s5|user1已添加第1个设备并共享给user2；user1已取消共享，user2不再具有控制权限。|
|s6|user1已添加第1个设备并共享给user2；设备被移除后，user2无可控设备。|
|s7|user1先添加并分享第1个设备后将其移除，又添加了第2个设备；当前user2未被共享，无法控制设备。|
|s8|user1先添加并分享第1个设备后移除，又添加第2个设备并直接共享给user2；user2因自动获得权限可再次控制设备。|
