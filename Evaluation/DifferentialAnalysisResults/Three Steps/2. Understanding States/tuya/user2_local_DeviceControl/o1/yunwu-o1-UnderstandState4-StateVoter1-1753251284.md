# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已成功添加第1个设备；user2无任何权限。|
|2|错误状态|
|3|user1已成功添加第1个设备并直接共享给user2；user2自动获得该设备的控制权限，非家庭成员。|

# Divergent Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已成功添加第1个设备；user2无控制权限。|
|2|错误状态|
|3|user1已成功添加第1个设备并直接共享给user2；user2自动获得该设备的控制权限。|
|4|user1已成功添加第1个设备并已分享；user2当前成功控制该设备。|
|5|user1仍拥有第1个设备；已撤销对user2的分享，user2失去控制权限。|
|6|user1移除设备后当前无设备；user2也无任何控制权限。|
|7|user1再次添加第2个设备；user2无控制权限。|
|8|user1在第2个设备上再次执行分享；user2自动获得该设备的控制权限。|
