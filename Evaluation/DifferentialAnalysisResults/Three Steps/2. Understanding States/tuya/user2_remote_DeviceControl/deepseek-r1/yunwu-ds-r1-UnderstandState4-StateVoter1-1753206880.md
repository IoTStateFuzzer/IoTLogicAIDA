# Base Model
|状态|语义描述|
|-----|---------|
|S0|初始状态|
|S1|user1已添加设备（第1个实例）；user2未收到邀请，无控制权限。|
|S2|错误状态|
|S3|user1已添加设备（第1个实例）并直接共享设备给user2；user2具有控制权限（直接共享方式）。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|S0|初始状态|
|S1|user1已添加设备（第1个实例）；user2未收到邀请，无控制权限。|
|S2|错误状态|
|S3|user1已添加设备（第1个实例）并直接共享设备给user2；user2具有控制权限（直接共享方式）。|
|S4|user1已添加设备（第1个实例）并共享设备；user2具有控制权限（已执行远程操作）。|
|S5|user1已添加设备（第1个实例），但取消了共享；user2无控制权限。|
|S6|user1移除了设备；user2的控制权限因设备移除而失效。|
