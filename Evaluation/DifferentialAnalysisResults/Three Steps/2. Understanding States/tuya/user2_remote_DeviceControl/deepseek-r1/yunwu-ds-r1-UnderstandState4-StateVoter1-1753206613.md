# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加第一个设备实例；user2未收到邀请，无家庭成员身份，无控制权限。|
|2|错误状态|
|3|user1已添加第一个设备实例并直接共享设备；user2具有直接共享设备控制权限，但不是家庭成员。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加第一个设备实例；user2未收到邀请，无家庭成员身份，无控制权限。|
|2|错误状态|
|3|user1已添加第一个设备实例并直接共享设备；user2具有直接共享设备控制权限，但不是家庭成员。|
|4|user1已添加设备并共享；user2有直接共享设备控制权限（权限有效），但不是家庭成员。|
|5|user1已添加设备但取消共享；user2无控制权限，不是家庭成员。|
|6|user1已移除设备；user2无控制权限，不是家庭成员。|
