# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一次实例）；user2未收到邀请，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第一次实例）并已共享设备；user2已同意邀请，非家庭成员，有控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一次实例）；user2未收到邀请，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第一次实例）并已共享设备；user2已直接获得权限（非家庭成员），有控制权限。|
|4|user1设备已添加并共享；user2有控制权限（非家庭成员）。|
|5|user1设备已添加但已取消共享；user2权限被撤销，无控制权限。|
|6|user1设备已移除；user2权限失效，无控制权限。|
