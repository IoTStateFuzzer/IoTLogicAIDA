# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态|
|s1|user1已添加设备（第一个实例），未共享；user2未收到邀请，无权限。|
|s2|错误状态|
|s3|user1已添加设备（第一个实例），已共享设备；user2具有控制权限（直接共享，无需接受）。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态|
|s1|user1已添加设备（第一个实例），未共享；user2未收到邀请，无权限。|
|s2|错误状态|
|s3|user1已添加设备（第一个实例），已共享设备；user2具有控制权限（直接共享，无需接受）。|
|s4|user1已添加设备（第一个实例），已共享设备；user2具有控制权限（直接共享，无需接受）。|
|s5|user1已添加设备（第一个实例），共享已取消；user2权限被撤销，无控制权限。|
|s6|user1已移除设备；user2权限失效，无控制权限。|
