# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一实例）；user2未收到邀请，无权限。|
|2|错误状态|
|3|user1已添加设备（第一实例）并共享设备；user2有控制权限（直接共享）。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一实例）；user2未收到邀请，无权限。|
|2|错误状态|
|3|user1已添加设备（第一实例）并共享设备；user2有控制权限（直接共享）。|
|4|user1已添加设备（第一实例）并共享设备；user2有控制权限（直接共享）。|
|5|user1已添加设备（第一实例）但未共享（取消共享）；user2无权限。|
|6|user1已移除设备；user2无权限。|
