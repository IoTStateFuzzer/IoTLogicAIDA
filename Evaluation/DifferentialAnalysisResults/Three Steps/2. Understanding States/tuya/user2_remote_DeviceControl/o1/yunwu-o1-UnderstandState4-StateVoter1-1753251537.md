# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备实例；user2尚未获得任何权限。|
|2|错误状态|
|3|user1已添加第1个设备实例并直接共享给user2；user2自动获得设备控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加第1个设备实例；user2尚未获得任何权限。|
|2|错误状态|
|3|user1已添加第1个设备实例并直接共享给user2；user2自动获得设备控制权限。|
|4|user1已添加第1个设备实例，且已共享给user2；user2正在使用设备控制权限。|
|5|user1取消了对user2的共享；user2已失去该设备的控制权限。|
|6|user1移除了设备，当前无设备可用；user2也失去该设备的控制权限。|
