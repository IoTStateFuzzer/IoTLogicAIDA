# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已成功添加第1个设备；user2未被邀请，无控制权限。|
|2|错误状态|
|3|user1已添加第1个设备并共享给user2；由于无需手动接受，user2已获得该设备的控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已成功添加第1个设备；user2未被邀请，无控制权限。|
|2|错误状态|
|3|user1已添加第1个设备并分享给user2；user2自动获得该设备控制权限。|
|4|user2已成功控制被分享的设备；user1仍保有设备所有权。|
|5|user1取消了对设备的共享；user2失去该设备控制权限。|
|6|user1移除了设备；user2对该设备的权限随之失效。|
