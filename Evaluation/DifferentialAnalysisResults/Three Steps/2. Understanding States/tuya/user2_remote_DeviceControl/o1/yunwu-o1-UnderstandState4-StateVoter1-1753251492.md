# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2暂无控制权限。|
|2|错误状态|
|3|user1已添加第1个设备并直接共享给user2；user2已获得该设备的控制权限。|

# Divergent Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2暂无控制权限。|
|2|错误状态|
|3|user1已添加第1个设备并共享给user2；user2已获得该设备控制权限。|
|4|user1已添加第1个设备并共享给user2；user2已成功控制设备。|
|5|user1已添加第1个设备并共享给user2；现由user1取消共享，user2不再具有控制权限。|
|6|user1已添加第1个设备并共享给user2；现已移除设备，user2也不再具有控制权限。|
