# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2暂无任何控制权限。|
|2|错误状态|
|3|user1已添加第1个设备，并通过直接共享方式邀请user2；user2尚未接受邀请，暂无控制权限。|
|4|user1已添加第1个设备，并已向user2共享；user2成功接受邀请，现拥有该设备的控制权限。|

# Divergent Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2尚无控制权限。|
|2|错误状态|
|3|user1已添加第1个设备并已向user2共享；user2尚未接受邀请，暂无控制权限。|
|4|user1已添加第1个设备并已向user2共享；user2已接受邀请，拥有设备控制权限。|
|5|user1已添加第1个设备并已向user2共享（user2已接受），现取消了共享；user2不再拥有控制权限。|
|6|user1移除已添加的设备；user2此前虽已接受共享，但设备已被移除，无法再控制。|
|7|user1先后添加并共享给user2（已接受）后又取消共享，再次发起新的共享；user2尚未接受最新邀请，暂时无控制权限。|
