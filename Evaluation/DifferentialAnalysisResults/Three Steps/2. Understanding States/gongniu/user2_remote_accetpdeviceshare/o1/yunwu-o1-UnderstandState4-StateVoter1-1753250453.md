# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第一个设备；user2尚未被邀请，无任何权限。|
|2|错误状态|
|3|user1已添加第一个设备并进行了直接共享操作；user2收到邀请但尚未接受，无控制权限。|
|4|user1已添加第一个设备并进行了直接共享操作；user2已手动接受共享，拥有设备控制权限。|

# Divergent Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第一个设备；user2没有收到邀请，无权限。|
|2|错误状态|
|3|user1已添加第一个设备并进行了直接共享操作；user2尚未接受共享，无控制权限。|
|4|user1已添加并共享设备；user2已手动接受共享，拥有该设备控制权限。|
|5|user1取消了对设备的直接共享；user2失去控制权限，设备仍在user1名下。|
|6|user1移除了该设备；user2的直接共享权限随设备移除而失效，user1当前无设备。|
|7|在取消共享后，user1重新发起了共享操作；user2尚未再次接受邀请，暂无法控制该设备。|
