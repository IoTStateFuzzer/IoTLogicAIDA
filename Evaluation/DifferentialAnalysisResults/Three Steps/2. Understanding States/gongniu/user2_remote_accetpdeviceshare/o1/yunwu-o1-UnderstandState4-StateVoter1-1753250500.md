# Base Model
|状态|语义描述|
|---|---|
|s0|初始状态|
|s1|user1已添加第1个设备；user2暂无控制权限。|
|s2|错误状态|
|s3|user1已添加第1个设备并发起共享；user2已收到邀请但尚未接受，无控制权限。|
|s4|user1已添加第1个设备并共享；user2已接受共享，现具备控制权限。|

# Divergent Model
|状态|语义描述|
|---|---|
|s0|初始状态|
|s1|user1已添加第1个设备；user2无共享邀请，不具备控制权限。|
|s2|错误状态|
|s3|user1已添加第1个设备并发起共享；user2尚未接受邀请，无控制权限。|
|s4|user1已添加第1个设备并共享；user2已接受邀请，获得控制权限。|
|s5|user1已添加第1个设备并先后共享、撤销共享；user2因被取消共享而无控制权限。|
|s6|user1已添加第1个设备并完成共享；随后user1移除设备，user2无归属设备，故无控制权限。|
|s7|user1已添加第1个设备并进行了先共享后撤销，再次共享；user2未再接受邀请，暂不具备控制权限。|
