# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加设备（首次添加），未执行共享操作；user2未收到共享邀请，无控制权限。|
|s2|错误状态。|
|s3|user1已添加设备（首次添加）并成功共享设备；user2已接收到共享邀请但尚未接受，暂不具备设备控制权限。|
|s4|user1已添加设备（首次添加）且已共享设备；user2已接受共享邀请，获得设备控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加设备（首次添加），未进行共享操作；user2未收到邀请，暂无控制权限。|
|s2|错误状态。|
|s3|user1已添加设备并成功共享设备；user2已接收到共享邀请，但未接受，暂未获得控制权限。|
|s4|user1已添加设备并共享设备；user2已接受共享邀请，获得设备控制权限。|
|s5|user1已添加设备，先共享后撤销共享；user2虽曾接受邀请，但因共享被取消而失去控制权限。|
|s6|user1已添加并共享设备，但随后移除设备；user2尽管之前接受过邀请，但设备移除后权限自动失效。|
|s7|user1已添加设备，先共享、后撤销共享再重新共享；user2原有的接受状态被撤销，需重新接受新邀请才能获得控制权限。|
