# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已成功添加设备（第1个实例）；user2未收到共享邀请，暂无控制权限。|
|s2|错误状态。|
|s3|user1已成功添加设备（第1个实例）并发起共享邀请；user2已收到邀请但未接受，暂未获得控制权限。|
|s4|user1已成功添加设备（第1个实例）并发起共享邀请；user2已接受邀请，成为家庭成员，从而拥有设备控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已成功添加设备（第1个实例）；user2未收到共享邀请，暂无控制权限。|
|s2|错误状态。|
|s3|user1已成功添加设备（第1个实例）并发起共享邀请；user2收到邀请但尚未接受，未获得控制权限。|
|s4|user1已成功添加设备（第1个实例）并发起共享邀请；user2接受邀请并获得家庭成员身份，从而拥有设备控制权限。|
|s5|user1在添加设备和共享后执行了撤销共享操作；user2虽然之前接受邀请获得控制权限，但在共享撤销后失去了该权限。|
|s6|user1在添加设备和共享后移除了设备；user2即便之前接受了共享邀请，也因设备移除而自动失去控制权限。|
|s7|user1先共享后撤销，再重新共享设备（设备仍为第1个实例）；user2之前的接受状态随撤销失效，且新共享邀请已发出但未被重新接受，故无控制权限。|
