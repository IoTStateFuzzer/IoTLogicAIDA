# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已成功添加设备（设备实例1）；user2尚未获得任何控制权限。|
|s2|错误状态。|
|s3|user1添加设备（设备实例1）后发起了共享，但user2尚未接受共享邀请，故暂未获得控制权限。|
|s4|user1添加设备（设备实例1）并发起共享，随后user2接受邀请，获得设备控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已成功添加设备（设备实例1）；user2尚未获得任何控制权限。|
|s2|错误状态。|
|s3|user1在添加设备（设备实例1）后发起了共享，但user2尚未接受共享邀请，故没有控制权限。|
|s4|user1添加设备（设备实例1）并发起共享，随后user2接受邀请获得设备的控制权限。|
|s5|user1添加设备（设备实例1）并发起共享，user2接受邀请后，user1执行取消共享，导致user2失去控制权限。|
|s6|user1添加设备（设备实例1）并发起共享、user2接受邀请后，user1移除设备，因设备已移除，user2的控制权限变为无效。|
|s7|user1添加设备（设备实例1），共享邀请被user2接受后被取消，再重新发起共享，但新共享未被user2再次接受，因此user2当前未获得控制权限。|
