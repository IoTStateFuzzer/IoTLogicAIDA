# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加第1个设备实例；user2未收到任何共享邀请，未获得控制权限。|
|s2|错误状态。|
|s3|user1已添加第1个设备实例并通过直接共享方式发起共享；user2已收到共享邀请但尚未手动接受，控制权限未生效。|
|s4|user1已添加第1个设备实例并通过直接共享方式发起共享；user2手动接受邀请后获得该设备实例的控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加第1个设备实例；user2未收到共享邀请，暂无控制权限。|
|s2|错误状态。|
|s3|user1已添加第1个设备实例并发起直接共享；user2已收到共享邀请但还未手动接受，控制权限未生效。|
|s4|user1已添加第1个设备实例并发起直接共享；user2手动接受后获得该设备的控制权限。|
|s5|user1在设备已共享、user2已接受后执行取消共享操作；共享权限被撤销，user2失去对该设备的控制权限。|
|s6|user1在设备已共享且user2已接受后移除设备实例；设备不存在共享配置，user2相应失去权限。|
|s7|user1先共享设备且user2已接受，再取消共享后重新发起共享；由于重新共享需要user2重新手动接受，目前user2处于待接受状态，尚未获得控制权限。|
