# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加设备（第一实例）；user2未收到共享邀请，无控制权限。|
|s2|错误状态。|
|s3|user1已添加设备（第一实例）并发起设备共享，发送了共享邀请；user2收到共享邀请但未手动接受，暂时无控制权限。|
|s4|user1已添加设备（第一实例）并发起设备共享；user2已接受共享邀请，获得了该设备实例的控制权限（直接共享设备方式）。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已成功添加设备（第一实例）；user2未收到共享邀请，无控制权限。|
|s2|错误状态。|
|s3|user1已添加设备并成功发起共享邀请；user2已收到共享邀请但尚未接受，暂时无控制权限。|
|s4|user1已添加设备并发起共享邀请；user2已接受共享邀请，因此获得该设备实例的控制权限（直接共享设备方式）。|
|s5|user1在共享成功后主动撤销了共享；尽管user2曾接受共享，但其控制权限已被撤销。|
|s6|user1在共享成功后移除了设备实例；设备移除使共享自动失效，user2失去控制权限。|
|s7|user1在添加设备、共享并获得user2接受后撤销共享，再次发起共享；user2的先前接受被撤销，且新共享尚待接受，因而目前无控制权限。|
