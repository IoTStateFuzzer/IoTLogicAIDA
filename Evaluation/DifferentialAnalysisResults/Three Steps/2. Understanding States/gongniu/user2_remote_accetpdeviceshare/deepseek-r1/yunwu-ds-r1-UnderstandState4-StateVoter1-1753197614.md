# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态|
|s1|user1已成功添加设备（第1次）；user2未收到邀请，无权限。|
|s2|错误状态|
|s3|user1已添加设备并共享；user2已收到邀请但未同意，无控制权限。|
|s4|user1已添加设备并共享；user2已接受邀请，拥有控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态|
|s1|user1已成功添加设备（第1次）；user2未收到邀请，无权限。|
|s2|错误状态|
|s3|user1已添加设备（第1次）并共享；user2已收到邀请但未同意，无控制权限。|
|s4|user1已添加设备（第1次）并共享；user2已接受邀请，拥有控制权限。|
|s5|user1已添加设备（第1次），但取消共享；user2权限被撤销，无控制权限。|
|s6|user1移除设备，无设备添加；user2权限因设备移除失效，无控制权限。|
|s7|user1已添加设备（第1次），取消共享后重新共享；user2收到新邀请但未同意，无控制权限。|
