# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一个实例）；user2未收到邀请。|
|2|错误状态|
|3|user1已添加并共享设备（第一个实例）；user2收到邀请但未同意。|
|4|user1已添加并共享设备（第一个实例）；user2已同意邀请并具有控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一个实例）；user2未收到邀请。|
|2|错误状态|
|3|user1已添加并共享设备（第一个实例）；user2收到邀请但未同意。|
|4|user1已添加并共享设备（第一个实例）；user2已同意邀请并具有控制权限。|
|5|user1已添加设备（第一个实例）；当前未共享；user2不再具有控制权限（权限已撤销）。|
|6|user1已移除设备；user2不再具有控制权限（因设备移除权限自动失效）。|
|7|user1已添加设备（第一个实例）并再次共享；user2收到新邀请但未同意。|
