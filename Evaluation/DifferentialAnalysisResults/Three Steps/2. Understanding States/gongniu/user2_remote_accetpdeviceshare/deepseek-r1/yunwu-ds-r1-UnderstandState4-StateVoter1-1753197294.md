# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已成功添加设备（第一次）；user2未收到邀请，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第一次）并直接共享；user2已收到邀请但未同意，无控制权限。|
|4|user1已添加设备（第一次）并直接共享；user2已同意邀请，具有控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一次）；user2未收到邀请，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第一次）并直接共享；user2已收到邀请但未同意，无控制权限。|
|4|user1已添加设备（第一次）并直接共享；user2已同意邀请，具有控制权限。|
|5|user1已添加设备（第一次）但已取消共享；user2的控制权限被撤销。|
|6|user1已移除设备；user2的控制权限因设备移除而失效。|
|7|user1已添加设备（第一次）并再次直接共享；user2已收到新邀请但未同意，无控制权限。|
