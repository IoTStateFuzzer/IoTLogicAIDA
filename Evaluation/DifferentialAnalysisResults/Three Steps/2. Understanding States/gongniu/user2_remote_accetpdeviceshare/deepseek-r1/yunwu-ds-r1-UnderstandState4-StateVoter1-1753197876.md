# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态|
|s1|user1已成功添加设备（第1个实例）；user2未收到邀请，无控制权限。|
|s2|错误状态|
|s3|user1已添加设备（第1个实例）并共享设备；user2收到邀请未同意，不是家庭成员，无控制权限。|
|s4|user1已添加设备（第1个实例）并共享设备；user2已接受邀请，不是家庭成员，拥有控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态|
|s1|user1已添加设备（第1个实例）；user2未收到邀请，不是家庭成员，无控制权限。|
|s2|错误状态|
|s3|user1已添加设备（第1个实例）并直接共享设备；user2收到邀请未同意，不是家庭成员，无控制权限。|
|s4|user1已添加设备（第1个实例）并直接共享设备；user2已接受邀请，不是家庭成员，拥有控制权限。|
|s5|user1已添加设备（第1个实例）但取消共享该设备；user2的权限被撤销，不是家庭成员，无控制权限。|
|s6|user1已移除设备（第1个实例被移除）；user2的权限因设备移除自动失效，不是家庭成员，无控制权限。|
|s7|user1已添加设备（第1个实例）并再次直接共享设备；user2收到新邀请未同意，不是家庭成员，无控制权限。|
