# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一实例），但未共享；user2未收到邀请，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第一实例）并成功共享；user2收到邀请但未同意，无控制权限。|
|4|user1已添加设备（第一实例）并成功共享；user2已同意邀请，有控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一实例），但未共享；user2未收到邀请，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第一实例）并成功共享；user2收到邀请但未同意，无控制权限。|
|4|user1已添加设备（第一实例）并成功共享；user2已同意邀请，有控制权限。|
|5|user1已添加设备（第一实例），但取消共享；user2权限被撤销，无控制权限。|
|6|user1已移除设备；user2权限失效，无控制权限。|
|7|user1已添加设备（第一实例），取消共享后再次共享；user2收到新邀请但未同意，无控制权限。|
