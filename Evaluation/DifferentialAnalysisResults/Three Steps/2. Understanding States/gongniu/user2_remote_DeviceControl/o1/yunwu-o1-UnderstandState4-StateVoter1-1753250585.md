# Base Model
|状态|语义描述|
|---|---|
|s0|初始状态|
|s1|user1已成功添加设备；user2尚未获得任何邀请。|
|s2|错误状态|
|s3|user1已添加设备并向user2发出共享请求；user2尚未接受邀请，暂无控制权限。|
|s4|user1已添加设备并成功共享给user2；user2已手动接受共享获得该设备的控制权限。|

# Divergent Model
|状态|语义描述|
|---|---|
|s0|初始状态|
|s1|user1已成功添加设备；user2尚未获得任何邀请。|
|s2|错误状态|
|s3|user1已成功添加设备，并向user2发出共享邀请；user2尚未接受邀请，暂无控制权限。|
|s4|user1已添加设备并成功共享给user2；user2已手动接受邀请并获得该设备的控制权限。|
|s5|user1已添加并共享设备给user2；user2已接受共享并成功进行设备操作。|
|s6|user1中止对user2的共享；user2的设备控制权限被撤销。|
|s7|user1再次向user2发出共享邀请；user2尚未手动接受，新邀请的控制权限尚未生效。|
|s8|user1移除设备；此设备已不存在，user2无法再控制。|
