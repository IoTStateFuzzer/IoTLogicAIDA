# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2尚无任何权限。|
|2|错误状态|
|3|user1已添加第1个设备并向user2发送设备共享邀请；user2尚未接受，无控制权限。|
|4|user1已添加第1个设备并共享；user2手动接受邀请后拥有控制权限。|

# Divergent Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2尚无任何权限。|
|2|错误状态|
|3|user1已添加第1个设备并共享给user2；user2尚未接受，暂无控制权限。|
|4|user1已添加第1个设备并共享；user2手动接受后获得设备控制权限。|
|5|user2使用共享权限进行设备控制；当前user2拥有设备控制权限。|
|6|user1取消了共享；user2的设备控制权限已被撤销。|
|7|user1再次发起设备共享；user2尚未接受新邀请，暂无设备控制权限。|
|8|user1移除设备；设备被删除后，user2不具备任何控制权限。|
