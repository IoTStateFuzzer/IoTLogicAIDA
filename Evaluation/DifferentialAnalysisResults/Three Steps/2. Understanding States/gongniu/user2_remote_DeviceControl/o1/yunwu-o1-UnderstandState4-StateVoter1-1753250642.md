# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2尚未被邀请，无控制权。|
|2|错误状态|
|3|user1已添加第1个设备并成功发起共享；user2已收到邀请但未接受，无控制权。|
|4|user1已添加第1个设备并成功共享；user2已接受邀请，拥有控制权。|

# Divergent Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2尚未被邀请，无控制权。|
|2|错误状态|
|3|user1已添加第1个设备并成功发起共享；user2已收到邀请但未接受，无控制权。|
|4|user1已添加第1个设备并成功共享；user2已接受邀请，拥有控制权。|
|5|user1已添加第1个设备并共享成功；user2已接受邀请且正在控制设备。|
|6|user1已添加第1个设备；user1取消了共享，user2不再具有控制权。|
|7|user1已添加第1个设备并重新发起共享；user2收到新邀请但尚未接受，无控制权。|
|8|user1已移除设备；user2的共享权限因此失效。|
