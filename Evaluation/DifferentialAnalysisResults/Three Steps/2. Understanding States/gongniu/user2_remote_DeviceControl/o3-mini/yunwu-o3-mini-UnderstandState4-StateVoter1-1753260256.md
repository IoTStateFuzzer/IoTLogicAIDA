# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加第1个设备实例；user2未收到邀请，无控制权限。|
|s2|错误状态。|
|s3|user1已添加第1个设备实例并发起共享邀请；user2已收到邀请但未同意，暂时无控制权限。|
|s4|user1已添加第1个设备实例并共享设备；user2已同意邀请，成为家庭成员并获得控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加第1个设备实例；user2未收到邀请，无控制权限。|
|s2|错误状态。|
|s3|user1已添加设备并发起共享邀请；user2收到邀请但尚未接受，无控制权限。|
|s4|user1已添加设备并共享成功；user2接受邀请，成为家庭成员并获得控制权限。|
|s5|user1添加并共享设备，user2接受邀请后进一步执行设备控制，处于控制状态。|
|s6|user1在共享后撤销共享；即便user2曾控制设备，其权限已被撤销。|
|s7|user1撤销后重新共享设备；user2重新收到邀请但尚未接受，故无控制权限。|
|s8|user1移除设备实例；设备移除导致共享失效，user2失去控制权限。|
