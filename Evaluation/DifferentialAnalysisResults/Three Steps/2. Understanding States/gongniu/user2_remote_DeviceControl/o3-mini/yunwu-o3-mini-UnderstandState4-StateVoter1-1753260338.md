# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加第1个设备实例；user2未收到任何邀请，暂无设备控制权限。|
|s2|错误状态。|
|s3|user1已添加第1个设备实例并通过SharePlug共享了设备；user2已收到邀请但尚未接受，暂未获得控制权限。|
|s4|user1已添加第1个设备实例并通过SharePlug共享了设备；user2已接受邀请，具备设备控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加第1个设备实例；user2尚未收到邀请，无控制权限。|
|s2|错误状态。|
|s3|user1已添加设备并共享（发送邀请），但user2尚未接受邀请，未获得控制权限。|
|s4|user1已添加设备并共享；user2已接受设备共享，获得了控制权限。|
|s5|user1已添加设备并共享；user2在接受邀请后进行了设备控制操作，保持控制权限。|
|s6|user1在添加设备、共享并获得user2控制后发起撤销共享操作，导致user2失去控制权限；设备实例仍有效。|
|s7|user1在撤销共享后重新共享设备，刷新了共享状态；user2之前的接受无效，需重新接受才能恢复控制权限。|
|s8|user1在设备共享并完成控制操作后移除了设备；设备移除使user2的控制权限被同时撤销。|
