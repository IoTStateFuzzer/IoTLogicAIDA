# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1成功添加设备实例1；user2未收到邀请，无控制权限。|
|s2|错误状态。|
|s3|user1成功添加设备实例1并发送共享邀请；user2已接收到邀请但尚未接受，当前无控制权限。|
|s4|user1成功添加设备实例1并发送共享邀请；user2已接受共享邀请，成为家庭成员，从而具备设备控制权。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加设备实例1；user2未收到任何分享邀请，无控制权限。|
|s2|错误状态。|
|s3|user1已添加设备实例1并发起分享邀请；user2收到邀请但未接受，暂时无控制权限。|
|s4|user1已添加设备实例1并发起分享邀请；user2接受邀请成为家庭成员，从而获得设备控制权限。|
|s5|user1已添加设备实例1并发起分享邀请；user2接受邀请并进行了设备控制操作，维持控制权限。|
|s6|user1在设备实例1上撤销了分享；因此user2虽曾获得控制权限，但现已被取消。|
|s7|user1在设备实例1上重新发起分享邀请；user2未重新接受邀请，仍无控制权限。|
|s8|user1移除了设备实例1；设备移除导致直接共享权限自动失效，user2失去控制权限。|
