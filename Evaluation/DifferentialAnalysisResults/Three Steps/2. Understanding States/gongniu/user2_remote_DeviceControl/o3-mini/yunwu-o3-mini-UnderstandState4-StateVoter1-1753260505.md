# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加第一设备实例；user2未获得设备共享权限。|
|s2|错误状态。|
|s3|user1已添加第一设备实例并发起了设备共享；user2收到共享邀请但尚未接受，因此未获得控制权限。|
|s4|user1已添加第一设备实例并发起了设备共享；user2已接受邀请，成为家庭成员并获得设备控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已添加第一设备实例；user2尚未获得共享权限。|
|s2|错误状态。|
|s3|user1已添加第一设备实例并发起设备共享，但user2尚未接受邀请，未获得控制权限。|
|s4|user1已添加设备并共享；user2已接受共享邀请，从而获得该设备的控制权限。|
|s5|user1添加并共享了设备；user2在接受邀请后执行了设备控制操作，现处于主动控制状态。|
|s6|user1在设备已共享并被控制后撤销共享，导致user2的控制权限被取消，而设备仍处于添加状态。|
|s7|user1撤销共享后重新共享设备，但user2未重新接受邀请，因此仍无控制权限。|
|s8|user1移除了设备实例，直接使共享权限失效，user2不再拥有设备控制权限。|
