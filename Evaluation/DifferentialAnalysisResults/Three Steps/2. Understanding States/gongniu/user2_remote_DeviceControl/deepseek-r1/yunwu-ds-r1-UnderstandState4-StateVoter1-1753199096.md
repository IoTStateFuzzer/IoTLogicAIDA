# Base Model
|状态|语义描述|
|-----|---------|
|S0|初始状态|
|S1|user1已添加设备（第1次实例），未共享；user2未收到邀请，不是家庭成员，无控制权限。|
|S2|错误状态|
|S3|user1已添加设备（第1次实例），已共享设备；user2收到邀请但未同意，不是家庭成员，无控制权限。|
|S4|user1已添加设备（第1次实例），已共享设备；user2已同意邀请，不是家庭成员，具有控制权限（直接共享）。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|S0|初始状态|
|S1|user1已添加设备（第1次实例），未共享；user2未收到邀请，不是家庭成员，无控制权限。|
|S2|错误状态|
|S3|user1已添加设备（第1次实例），已共享设备；user2收到邀请但未同意，不是家庭成员，无控制权限。|
|S4|user1已添加设备（第1次实例），已共享设备；user2已同意邀请，不是家庭成员，具有控制权限（直接共享）。|
|S5|user1已添加设备（第1次实例），已共享设备；user2已同意邀请，不是家庭成员，具有控制权限（直接共享）。|
|S6|user1已添加设备（第1次实例），未共享（已取消共享）；user2权限已撤销，不是家庭成员，无控制权限。|
|S7|user1已添加设备（第1次实例），已重新共享设备；user2收到新邀请但未同意，不是家庭成员，无控制权限。|
|S8|user1已移除设备（无当前设备实例）；user2权限因设备移除而失效，不是家庭成员，无控制权限。|
