# Base Model
|状态|语义描述|
|-----|---------|
|S0|初始状态|
|S1|user1已成功添加设备（第1个实例）；user2未收到邀请，权限状态为“无控制权”。|
|S2|错误状态|
|S3|user1已成功添加设备（第1个实例）并直接共享设备；user2已收到邀请但未同意，权限状态为“无控制权”。|
|S4|user1已成功添加设备（第1个实例）并直接共享设备；user2已同意邀请，权限状态为“有控制权”。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|S0|初始状态|
|S1|user1已成功添加设备（第1个实例）；user2未收到邀请，权限状态为“无控制权”。|
|S2|错误状态|
|S3|user1已成功添加设备（第1个实例）并直接共享设备；user2已收到邀请但未同意，权限状态为“无控制权”。|
|S4|user1已成功添加设备（第1个实例）并直接共享设备；user2已同意邀请，权限状态为“有控制权”。|
|S5|user1已成功添加设备（第1个实例）并直接共享设备；user2已同意邀请且成功执行控制操作，权限状态为“有控制权”。|
|S6|user1设备（第1个实例）存在但已取消共享；user2权限被撤销，状态为“无控制权”。|
|S7|user1设备（第1个实例）存在并重新发起共享；user2收到新邀请但未同意，权限状态为“无控制权”。|
|S8|user1已移除设备（第1个实例）；user2权限因设备移除失效，状态为“无控制权”。|
