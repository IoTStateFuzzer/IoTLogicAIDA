# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态|
|s1|user1已添加设备；user2未收到邀请，无控制权。|
|s2|错误状态|
|s3|user1已添加设备并共享；user2收到邀请但未同意，无控制权。|
|s4|user1已添加设备并共享；user2已同意邀请，具有控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态|
|s1|user1已添加设备；user2未收到邀请，无控制权。|
|s2|错误状态|
|s3|user1已添加设备并共享；user2收到邀请但未同意，无控制权。|
|s4|user1已添加设备并共享；user2已同意邀请，具有控制权限。|
|s5|user1已添加设备并共享；user2已同意邀请，具有控制权限。|
|s6|user1已添加设备；已取消共享，user2无控制权。|
|s7|user1已添加设备并重新共享；user2收到新邀请但未同意，无控制权。|
|s8|user1已移除设备；user2无控制权（因设备被移除）。|
