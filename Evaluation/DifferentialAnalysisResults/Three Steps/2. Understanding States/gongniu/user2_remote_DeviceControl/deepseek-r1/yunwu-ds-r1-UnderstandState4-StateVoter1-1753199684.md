# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一个实例）；user2未收到邀请，无权限。|
|2|错误状态|
|3|user1已添加设备（第一个实例）并共享设备；user2收到邀请但未同意，无控制权。|
|4|user1已添加设备（第一个实例）并共享设备；user2已同意邀请，有控制权限（直接共享方式）。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一个实例）；user2未收到邀请，无权限。|
|2|错误状态|
|3|user1已添加设备（第一个实例）并共享设备；user2收到邀请但未同意，无控制权。|
|4|user1已添加设备（第一个实例）并共享设备；user2已同意邀请，有控制权限（直接共享方式）。|
|5|user1已添加设备（第一个实例）并共享设备；user2已同意邀请，有控制权限（直接共享方式）。|
|6|user1已添加设备（第一个实例）但已取消共享；user2无控制权限。|
|7|user1已添加设备（第一个实例）并重新共享设备；user2收到新邀请但未同意，无控制权。|
|8|user1未添加设备（设备已移除）；user2无权限。|
