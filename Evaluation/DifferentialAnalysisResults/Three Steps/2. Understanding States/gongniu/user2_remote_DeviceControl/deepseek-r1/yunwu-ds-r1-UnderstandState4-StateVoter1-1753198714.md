# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态|
|s1|user1已添加设备（第一次实例）；user2未收到邀请，无权限。|
|s2|错误状态|
|s3|user1已添加设备（第一次实例）并成功共享；user2收到邀请但未同意，无控制权限。|
|s4|user1已添加设备（第一次实例）并共享；user2已接受邀请，获得设备控制权。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态|
|s1|user1已添加设备（第一次实例）；user2未收到邀请，无权限。|
|s2|错误状态|
|s3|user1已添加设备（第一次实例）并共享；user2收到邀请但未同意，无控制权限。|
|s4|user1已添加设备（第一次实例）并共享；user2已接受邀请，获得设备控制权。|
|s5|user1已添加设备（第一次实例）并共享；user2拥有控制权限（已执行控制操作）。|
|s6|user1已添加设备（第一次实例）但取消共享；user2无控制权限（权限被撤销）。|
|s7|user1已添加设备（第一次实例）并重新共享；user2收到新邀请但未同意，无控制权限。|
|s8|user1已移除设备（第一次实例）；user2无控制权限（设备不存在）。|
