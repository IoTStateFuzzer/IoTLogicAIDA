# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2无控制权限。|
|2|错误状态|
|3|user1已添加第1个设备并向user2发起设备分享；user2未接受分享，无控制权限。|
|4|user1已添加第1个设备并向user2发起设备分享；user2已接受分享，拥有该设备的控制权限。|
|5|user1移除第1个设备，分享自动失效；user2失去该设备的控制权限。|
|6|user1重新添加第2个设备；尚未分享给user2，user2无控制权限。|
|7|user1添加第2个设备并向user2发起分享；user2未接受分享，无控制权限。|
|8|user1添加第2个设备并向user2发起分享；user2已接受分享，重新获得该设备的控制权限。|
