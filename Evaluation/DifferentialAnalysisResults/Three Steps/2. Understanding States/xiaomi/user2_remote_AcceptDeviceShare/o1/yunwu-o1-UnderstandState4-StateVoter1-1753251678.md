# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加1台设备；user2尚无控制权限。|
|2|错误状态|
|3|user1已添加1台设备并发起分享；user2已收到设备分享邀请但尚未接受。|
|4|user1已添加1台设备并发起分享；user2已手动接受分享并拥有设备控制权。|
|5|user1已添加并分享设备后又移除该设备；user2对已移除设备的控制权限已失效。|
|6|user1在移除第一台设备后添加了第二台设备；user2尚无控制权限。|
|7|user1为第二台设备发起分享；user2已收到分享邀请但尚未接受。|
|8|user1为第二台设备发起分享；user2已手动接受分享并获得该设备控制权。|
