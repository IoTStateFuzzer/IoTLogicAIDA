# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2无控制权。|
|2|错误状态|
|3|user1已添加第1个设备并发起共享；user2尚未接受，暂无控制权。|
|4|user1已添加第1个设备并完成共享；user2已接受，拥有控制权。|
|5|user1已移除第1个设备；因设备被移除，user2无控制权。|
|6|user1已重新添加第2个设备；user2无控制权。|
|7|user1已添加第2个设备并发起共享；user2尚未接受，暂无控制权。|
|8|user1已添加第2个设备并完成共享；user2已接受，拥有控制权。|
