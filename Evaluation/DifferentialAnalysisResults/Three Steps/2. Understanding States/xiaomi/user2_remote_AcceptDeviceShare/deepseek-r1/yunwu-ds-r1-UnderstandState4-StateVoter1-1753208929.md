# Base Model
|状态|语义描述|
|-----|---------|
|S0|初始状态|
|S1|user1已添加第一个设备实例，未共享；user2未收到邀请，无控制权限。|
|S2|错误状态|
|S3|user1已添加第一个设备实例并共享了设备；user2收到邀请但未同意，无控制权限。|
|S4|user1已添加第一个设备实例并共享了设备；user2已接受邀请，有控制权限。|
|S5|user1已添加第一个设备实例，共享后移除了设备；user2权限失效，无控制权限。|
|S6|user1已添加第二个设备实例，未共享；user2未收到邀请，无控制权限。|
|S7|user1已添加第二个设备实例并共享了设备；user2收到邀请但未同意，无控制权限。|
|S8|user1已添加第二个设备实例并共享了设备；user2已接受邀请，有控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|S0|初始状态|
|S1|user1已添加第一个设备实例，未共享；user2未收到邀请，无控制权限。|
|S2|错误状态|
|S3|user1已添加第一个设备实例并直接共享设备；user2收到邀请但未同意，无控制权限。|
|S4|user1已添加第一个设备实例并直接共享设备；user2已接受邀请，有控制权限。|
|S5|user1已移除设备；user2权限失效（直接共享终止），无控制权限。|
|S6|user1已添加第二个设备实例，未共享；user2未收到新邀请，无控制权限。|
|S7|user1已添加第二个设备实例并直接共享设备；user2收到新邀请但未同意，无控制权限。|
|S8|user1已添加第二个设备实例并直接共享设备；user2已接受新邀请，有控制权限。|
