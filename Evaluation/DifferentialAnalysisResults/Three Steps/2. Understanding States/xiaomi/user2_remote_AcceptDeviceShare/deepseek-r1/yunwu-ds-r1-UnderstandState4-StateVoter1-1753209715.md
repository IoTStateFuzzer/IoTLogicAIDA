# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态|
|s1|user1已添加设备（第一个实例）；user2未收到邀请，无控制权限。|
|s2|错误状态|
|s3|user1已添加设备（第一个实例）并直接共享给user2；user2收到邀请未同意，无控制权限。|
|s4|user1已添加设备（第一个实例）并直接共享给user2；user2已同意邀请，有控制权限。|
|s5|user1已添加设备（第一个实例）、共享并移除；user2权限因设备移除而失效。|
|s6|user1已添加设备（第二个实例）；user2未收到邀请，无控制权限（之前权限已失效）。|
|s7|user1已添加设备（第二个实例）并直接共享给user2；user2收到邀请未同意，无控制权限。|
|s8|user1已添加设备（第二个实例）并直接共享给user2；user2已同意邀请，有控制权限。|
