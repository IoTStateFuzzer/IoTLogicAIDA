# Base Model
|状态|语义描述|
|-----|---------|
|s0|user1未添加设备；user2未收到邀请，无权限。|
|s1|user1已成功添加设备（第一个实例）；user2未收到邀请，无权限。|
|s2|错误状态|
|s3|user1已添加设备（第一个实例）并共享；user2收到邀请未同意，无控制权限。|
|s4|user1已添加设备（第一个实例）并共享；user2已同意邀请，有控制权限（基于直接共享）。|
|s5|user1已移除设备；user2的邀请已失效，无权限。|
|s6|user1已重新添加设备（第二个实例）；user2未收到邀请，无权限。|
|s7|user1已重新添加设备并共享（第二个实例）；user2收到邀请未同意，无控制权限。|
|s8|user1已重新添加设备并共享（第二个实例）；user2已同意邀请，有控制权限（基于直接共享）。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态|
|s1|user1已添加设备（第一个实例）；user2未收到邀请，无权限。|
|s2|错误状态|
|s3|user1已添加设备（第一个实例）并直接共享；user2收到邀请未同意，无控制权限。|
|s4|user1已添加设备（第一个实例）并直接共享；user2已同意邀请，有控制权限（直接共享方式）。|
|s5|user1已移除设备；user2的权限因设备移除自动失效，无权限。|
|s6|user1已重新添加设备（第二个实例）；user2未收到新邀请，无权限。|
|s7|user1已重新添加设备并直接共享（第二个实例）；user2收到新邀请未同意，无控制权限。|
|s8|user1已重新添加设备并直接共享（第二个实例）；user2已同意新邀请，有控制权限（直接共享方式）。|
