# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已成功添加设备（第1次实例）；user2未收到邀请，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第1次实例）并共享；user2收到邀请但未同意，无控制权限。|
|4|user1已添加设备（第1次实例）并共享；user2已同意邀请，有控制权限。|
|5|user1设备已移除；user2未同意邀请，无控制权限。|
|6|user1已添加设备（第2次实例）；user2未收到邀请，无控制权限。|
|7|user1已添加设备（第2次实例）并共享；user2收到邀请但未同意，无控制权限。|
|8|user1已添加设备（第2次实例）并共享；user2已同意邀请，有控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第1次实例）；user2不是家庭成员，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第1次实例）并发送家庭邀请；user2已收到邀请但未接受，无控制权限。|
|4|user1已添加设备（第1次实例）并发送家庭邀请；user2已接受邀请成为家庭成员，有控制权限。|
|5|user1设备已移除；user2是家庭成员，无控制权限（因设备不存在）。|
|6|user1已添加设备（第2次实例）；user2是家庭成员，有控制权限（自动获得）。|
|7|user1已添加设备（第2次实例）并再次发送邀请（冗余操作）；user2是家庭成员，有控制权限。|
|8|user1已添加设备（第2次实例）并再次发送邀请；user2再次接受邀请（冗余操作）；user2是家庭成员，有控制权限。|
