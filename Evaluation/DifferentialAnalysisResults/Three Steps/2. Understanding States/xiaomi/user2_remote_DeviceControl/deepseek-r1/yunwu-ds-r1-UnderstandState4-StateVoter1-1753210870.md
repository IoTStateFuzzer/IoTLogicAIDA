# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1 已添加第一个设备实例；未共享给 user2，user2 无权限。|
|2|错误状态|
|3|user1 已添加第一个设备实例并共享给 user2；user2 收到邀请但未同意，无控制权限。|
|4|user1 已添加第一个设备实例并共享给 user2；user2 已同意邀请，具有控制权限。|
|5|user1 移除了设备；由于直接共享，user2 的权限已失效。|
|6|user1 已添加第二个设备实例；未共享给 user2，user2 无权限。|
|7|user1 已添加第二个设备实例并共享给 user2；user2 收到邀请但未同意，无控制权限。|
|8|user1 已添加第二个设备实例并共享给 user2；user2 已同意邀请，具有控制权限。|
