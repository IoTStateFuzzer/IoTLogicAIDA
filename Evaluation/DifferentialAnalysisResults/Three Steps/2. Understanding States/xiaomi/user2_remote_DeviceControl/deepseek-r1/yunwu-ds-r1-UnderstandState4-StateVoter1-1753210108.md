# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加第一个设备实例；user2未收到邀请，权限状态为“无控制权”。|
|2|错误状态|
|3|user1已添加第一个设备实例并共享；user2收到邀请未同意，权限状态为“无控制权”。|
|4|user1已添加第一个设备实例并共享；user2已同意邀请，权限状态为“有控制权”。|
|5|user1已移除设备；user2权限状态为“无控制权”（设备移除后权限失效）。|
|6|user1已添加第二个设备实例；user2未收到邀请，权限状态为“无控制权”。|
|7|user1已添加第二个设备实例并共享；user2收到邀请未同意，权限状态为“无控制权”。|
|8|user1已添加第二个设备实例并共享；user2已同意邀请，权限状态为“有控制权”。|
