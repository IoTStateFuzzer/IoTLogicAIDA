# Base Model
|状态|语义描述|
|-----|---------|
|S0|初始状态|
|S1|user1已成功添加设备（第一次）；user2未收到邀请，无权限。|
|S2|错误状态|
|S3|user1已添加设备（第一次）并直接共享；user2收到邀请但未同意，无权限。|
|S4|user1已添加设备（第一次）并直接共享；user2已接受邀请，拥有控制权限（基于设备实例）。|
|S5|user1已移除设备；user2的邀请因设备移除而失效，无权限。|
|S6|user1已重新添加设备（第二次）；user2无新邀请，无权限。|
|S7|user1已添加设备（第二次）并重新直接共享；user2收到新邀请但未同意，无权限。|
|S8|user1已添加设备（第二次）并直接共享；user2已接受邀请，拥有控制权限（基于设备实例）。|
