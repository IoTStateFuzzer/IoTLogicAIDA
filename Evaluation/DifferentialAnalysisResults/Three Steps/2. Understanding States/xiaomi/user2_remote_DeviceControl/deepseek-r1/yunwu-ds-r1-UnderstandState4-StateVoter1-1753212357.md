# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第一次）；user2未收到邀请，无控制权限。|
|2|错误状态|
|3|user1已添加设备（第一次）并共享；user2收到邀请但未同意，无控制权限。|
|4|user1已添加设备（第一次）并共享；user2已接受邀请，有控制权限（直接共享）。|
|5|user1已移除设备；user2无控制权限（邀请因设备移除失效）。|
|6|user1已添加设备（第二次）；未共享；user2未收到邀请，无控制权限。|
|7|user1已添加设备（第二次）并共享；user2收到邀请但未同意，无控制权限。|
|8|user1已添加设备（第二次）并共享；user2已接受邀请，有控制权限（直接共享）。|
