# Base Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加设备（第1个实例）；user2未收到邀请，权限状态为“无权限”。|
|2|错误状态|
|3|user1已添加设备（第1个实例）并共享；user2已收到邀请但未同意，权限状态为“无控制权”。|
|4|user1已添加设备（第1个实例）并共享；user2已同意邀请，权限状态为“有控制权”。|
|5|user1已移除设备；user2的权限已失效（由于设备移除）。|
|6|user1已添加设备（第2个实例）；user2未收到邀请，权限状态为“无权限”。|
|7|user1已添加设备（第2个实例）并共享；user2已收到邀请但未同意，权限状态为“无控制权”。|
|8|user1已添加设备（第2个实例）并共享；user2已同意邀请，权限状态为“有控制权”。|
