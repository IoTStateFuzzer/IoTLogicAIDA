# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2未被邀请或未接受，无控制权限。|
|2|错误状态|
|3|user1已添加第1个设备并共享给user2；user2尚未接受邀请，无控制权限。|
|4|user1已添加第1个设备并共享给user2；user2已接受共享，拥有该设备的控制权限。|
|5|user1添加并共享了第1个设备后又移除该设备；user2因未接受或设备已移除而无控制权限。|
|6|user1先添加并移除第1个设备，又重新添加了第2个设备；当前仅第2个设备存在，user2无控制权限。|
|7|user1在第2个设备上再次发起共享；user2尚未接受邀请，无控制权限。|
|8|user1在第2个设备上再次发起共享；user2已接受共享，拥有该设备的控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加第1个设备；user2未被邀请或未接受，无控制权限。|
|2|错误状态|
|3|user1已添加第1个设备并发起共享；user2尚未接受邀请，无控制权限。|
|4|user1已添加第1个设备并分享给user2；user2已接受共享，拥有该设备控制权限。|
|5|user1添加并共享第1个设备后又将其移除；当前设备不存在，user2无控制权限。|
|6|user1先添加并共享第1个设备后移除，再添加新设备；user2未被邀请，无控制权限。|
|7|user1在新添加的设备上再次发起共享；user2尚未接受邀请，无控制权限。|
|8|user1在新添加的设备上再次发起共享；user2已接受共享，拥有该设备控制权限。|
|9|user1已添加并共享第1个设备；user2已接受共享并正在控制该设备。|
|10|user1移除设备后，当前设备不存在；user2不再拥有控制权限。|
|11|user1移除第1个设备后又添加新的设备；user2未被邀请，无控制权限。|
|12|user1取消共享，user2不再拥有控制权限；设备仍存在。|
|13|user1再次对同一设备发起共享；user2尚未接受邀请，无控制权限。|
|14|user1移除第1个设备后又添加新设备并发起共享；user2尚未接受邀请，无控制权限。|
|15|user1再次取消并共享该设备后将其移除；当前设备不存在，user2无控制权限。|
|16|user1取消并再次共享后移除第1个设备，又添加新设备；user2未被邀请，无控制权限。|
|17|user1添加新设备并发起共享；user2尚未接受邀请，无控制权限。|
|18|user1在新设备上发起共享；user2已接受共享，拥有该设备控制权限。|
|19|user1移除第1个设备后添加第2个并共享；user2已接受共享并正在控制该设备。|
|20|user1对第2个设备取消共享；user2不再拥有控制权限。|
|21|user1再次对第2个设备发起共享；user2尚未接受邀请，无控制权限。|
|22|user1移除第1个设备又添加第2个并共享；user2已接受共享，拥有该设备控制权限。|
