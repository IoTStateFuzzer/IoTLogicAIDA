# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2暂无控制权限。|
|2|错误状态|
|3|user1已添加第1个设备并向user2发起直接共享；user2尚未接受，无控制权。|
|4|user1已添加第1个设备并向user2发起直接共享；user2已接受，拥有控制权。|
|5|user1已移除第1个设备；该设备的共享失效，user2无控制权。|
|6|user1重新添加了第2个设备；未对user2共享，user2无控制权。|
|7|user1已添加第2个设备并向user2发起直接共享；user2尚未接受，无控制权。|
|8|user1已添加第2个设备并向user2发起直接共享；user2已接受，拥有控制权。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|0|初始状态|
|1|user1已添加第1个设备；user2暂无控制权限。|
|2|错误状态|
|3|user1已添加第1个设备并向user2共享；user2尚未接受无控制权。|
|4|user1已添加第1个设备并向user2共享；user2已接受共享，拥有控制权。|
|5|user1已移除第1个设备；共享失效，user2无控制权。|
|6|user1已重新添加第2个设备；未共享给user2，user2无控制权。|
|7|user1已添加第2个设备并向user2共享；user2尚未接受无控制权。|
|8|user1已添加第2个设备并向user2共享；user2已接受共享，拥有控制权。|
|9|user1已添加第1个设备并向user2共享；user2已接受且正进行设备控制。|
|10|user1已移除第1个设备；user2失去控制权。|
|11|user1已重新添加新设备；尚未共享，user2无控制权。|
|12|user1已添加并共享给user2，user2曾经控制过；现user1取消共享，user2失去控制权。|
|13|user1取消共享后再次共享当前设备；user2尚未接受，无控制权。|
|14|user1移除旧设备后又添加新设备并共享；user2尚未接受，无控制权。|
|15|user1先前已取消并再次共享后最终移除设备；设备不存在，user2无控制权。|
|16|user1重新添加新设备；尚未共享，user2无控制权。|
|17|user1重新添加并共享设备；user2尚未接受，无控制权。|
|18|user1已共享新设备；user2已接受共享，拥有控制权。|
|19|user1移除旧设备后添加了新设备并共享；user2已接受且正在控制该设备。|
|20|user1已取消设备共享；user2失去控制权。|
|21|user1重新共享设备；user2尚未接受，无控制权。|
|22|user1移除旧设备后重新添加并共享给user2；user2已接受共享，拥有控制权。|
