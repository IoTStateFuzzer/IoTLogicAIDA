# Base Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2尚未被分享，无控制权。|
|2|错误状态|
|3|user1已添加第1个设备并向user2发起分享；user2尚未接受，无控制权。|
|4|user1已添加第1个设备并向user2发起分享；user2已接受，获得控制权。|
|5|user1已移除第1个设备；直接分享已失效，user2无控制权。|
|6|user1再次添加设备（第2次绑定）；user2未获得新设备的分享，无控制权。|
|7|user1为第2个设备发起分享；user2尚未接受，无控制权。|
|8|user1为第2个设备发起分享；user2已接受，获得控制权。|

# Divergent Model
|状态|语义描述|
|---|---|
|0|初始状态|
|1|user1已添加第1个设备；user2尚未被分享，无控制权。|
|2|错误状态|
|3|user1已添加第1个设备并发起分享；user2尚未接受，无控制权。|
|4|user1已添加第1个设备并发起分享；user2已接受，获得控制权。|
|5|user1已移除第1个设备；直接分享失效，user2无控制权。|
|6|user1再次添加第2个设备；user2未获任何分享，无控制权。|
|7|user1为第2个设备发起分享；user2尚未接受，无控制权。|
|8|user1为第2个设备发起分享；user2已接受，获得控制权。|
|9|user1已添加第1个设备并共享；user2已接受并正在使用设备（DeviceControl）。|
|10|user1移除第1个设备；user2分享失效，无控制权。|
|11|user1重新添加第2个设备；user2未获分享，无控制权。|
|12|user1仍有第1个设备并执行取消分享；user2权限被收回，无控制权。|
|13|user1重新发起分享第1个设备；user2尚未接受，无控制权。|
|14|user1移除第1个设备后添加第2个设备并发起分享；user2尚未接受，无控制权。|
|15|user1移除设备后无设备存在；user2无控制权。|
|16|user1重新添加第2个设备；user2未被分享，无控制权。|
|17|user1为第2个设备发起分享；user2尚未接受，无控制权。|
|18|user1为第2个设备发起分享；user2已接受，获得控制权。|
|19|user1移除第1个设备并添加第2个设备并分享；user2已接受并正在使用第2个设备。|
|20|user1取消分享第2个设备；user2不再拥有控制权。|
|21|user1再次分享第2个设备；user2尚未接受，无控制权。|
|22|user1先前移除第1个设备并添加第2个设备；user2已接受新分享并获得控制权。|
