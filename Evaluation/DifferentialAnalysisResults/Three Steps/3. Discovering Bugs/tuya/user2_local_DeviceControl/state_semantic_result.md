# Base Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已成功添加第1个设备实例；user2未被邀请或共享，因此无控制权限。|
|s2|错误状态。|
|s3|user1已添加第1个设备实例并执行共享操作；user2通过直接共享方式获得了设备控制权限。|

# Divergent Model
|状态|语义描述|
|-----|---------|
|s0|初始状态。|
|s1|user1已成功添加第1个设备实例；user2未获得共享权限，故无控制权限。|
|s2|错误状态。|
|s3|user1已添加第1个设备实例并共享设备；user2通过共享自动获得了设备控制权限。|
|s4|在第1个设备实例共享后，user2执行了设备控制操作，确认其拥有控制权限。|
|s5|user1在共享后取消了共享操作，但设备实例依然存在；因此user2的控制权限被撤销。|
|s6|user1在设备共享期间将设备实例移除；设备移除导致user2失去控制权限。|
|s7|user1先移除已共享的设备实例后重新添加设备，形成第2个设备实例；但未对新实例执行共享，故user2无控制权限。|
|s8|user1在添加第二个设备实例后进行了共享操作；因此user2通过共享获得了该新设备实例的控制权限。|
